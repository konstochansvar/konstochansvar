## How to Extract Images

1. Visit https://konstochansvar.se
2. Right-click on each image
3. Select "Save image as..." or "Inspect element" to find the image URL
4. Download and save to the appropriate directory
5. Ensure filenames match exactly (case-sensitive)

## Image Paths in Original Site

The original site uses paths like:
- `Bilder/ikalogo2.jpg` → `assets/images/icons/ikalogo2.jpg`
- `Bilder/trojajagarlikgiltig.JPG` → `assets/images/content/trojajagarlikgiltig.JPG`

## Notes

- All image paths in the content have been updated to use `/assets/images/` structure
- Image dimensions are preserved from the original site
